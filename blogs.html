My Daily Learning Journey

Learning Docker

I’ve started diving into Docker before moving on to Kubernetes. My focus right now is understanding containerization — how Docker packages applications with all dependencies so they run consistently across environments. I’m experimenting with building images, running containers, and using Docker Compose to manage multi-container setups.

This foundation will make Kubernetes much easier to grasp later, since Kubernetes orchestrates Docker containers at scale. For now, I’m excited to get hands-on with Docker and see how it simplifies deployment workflows.
